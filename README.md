# TBW Launcher

Кастомный лаунчер для Minecraft-сервера TBW.

## 📌 О проекте

TBW Launcher — это полностью кастомное решение для запуска клиента сервера TBW.  
Лаунчер разработан с нуля и не привязан к официальной инфраструктуре Minecraft.

## ⚙️ Особенности

- 🔐 Собственная система авторизации
- 📦 Загрузка и обновление клиента
- 🚫 Отсутствие привязки к лицензионным сервисам (скины, аккаунты и т.д.)

## 🚀 Установка и запуск

1. Скачайте лаунчер
2. Запустите исполняемый файл
3. Авторизуйтесь или создайте аккаунт
4. Установите нужную сборку
5. Заходите в игру

## ⚠️ Важно

Данный лаунчер предназначен **исключительно для использования с сервером TBW**.
